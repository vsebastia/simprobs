Explicación breve

El contraste chi-cuadrado para una varianza permite evaluar si la variabilidad de una población normal es compatible con un valor de referencia \(\sigma_0^2\). Es una herramienta habitual en control de calidad (¿ha aumentado la dispersión de un proceso?) y en la validación de especificaciones de precisión de instrumentos de medida.

La prueba supone que la población sigue una distribución normal y es especialmente sensible a desviaciones de este supuesto, más que el test t para la media. Para datos no normales existen alternativas más robustas.

Hipótesis y estadístico

\(H_0: \sigma^2 = \sigma_0^2\) (equivalentemente \(\sigma = \sigma_0\))

\(H_1: \sigma^2 \neq \sigma_0^2\), \(\sigma^2 > \sigma_0^2\) o \(\sigma^2 < \sigma_0^2\)

\(\chi^2 = \dfrac{(n-1)\,s^2}{\sigma_0^2} \sim \chi^2_{n-1}\) bajo \(H_0\)

\(IC_{(1-\alpha)}\text{ para }\sigma^2: \left[\dfrac{(n-1)\,s^2}{\chi^2_{1-\alpha/2,\,n-1}},\; \dfrac{(n-1)\,s^2}{\chi^2_{\alpha/2,\,n-1}}\right]\)

  • \(s\): desviación típica muestral.
  • \(\sigma_0\): desviación típica de referencia bajo \(H_0\).
  • \(n\): tamaño muestral.
  • \(gl = n - 1\): grados de libertad de la distribución chi-cuadrado.

Guía práctica de uso

  • Contraste bilateral: se usa cuando no se tiene una hipótesis direccional previa sobre si la varianza ha aumentado o disminuido.
  • Cola derecha (σ > σ₀): adecuado en control de calidad para detectar aumentos en la variabilidad de un proceso productivo o instrumento de medición.
  • Cola izquierda (σ < σ₀): útil para verificar si se ha conseguido reducir la variabilidad tras una mejora del proceso.

Si el p-valor es menor que \(\alpha\), se rechaza \(H_0\): la varianza es significativamente distinta de \(\sigma_0^2\). El intervalo de confianza para \(\sigma^2\) y \(\sigma\) complementa el resultado al mostrar el rango de valores plausibles.

Supuestos del test

  • La población debe seguir una distribución normal. El test chi-cuadrado para la varianza es muy sensible a la no normalidad.
  • Las observaciones deben ser independientes.
  • Para datos no normales considera alternativas robustas: test de Bartlett (variante más poderosa pero igualmente sensible a la normalidad), test de Levene o test de Fligner-Killeen (más robusto ante no normalidad).

Ejemplo resuelto

Un proceso industrial tiene una especificación de desviación típica de \(\sigma_0 = 10\) unidades. Se toma una muestra de \(n = 25\) piezas con desviación típica muestral \(s = 12\). Se contrasta \(H_0: \sigma^2 = 100\) frente a \(H_1: \sigma^2 \neq 100\) con \(\alpha = 0{,}05\).

\(\chi^2 = \dfrac{(25-1) \times 12^2}{10^2} = \dfrac{24 \times 144}{100} = 34{,}56\)

Con \(gl = 24\) grados de libertad y contraste bilateral, el p-valor es \(p \approx 0{,}076\). Como \(p = 0{,}076 > \alpha = 0{,}05\), no se rechaza \(H_0\): no hay evidencia estadística suficiente para concluir que la varianza del proceso es distinta de 100.

El IC 95% para \(\sigma^2\) es aproximadamente \([82{,}5;\; 239{,}0]\), que contiene el valor \(\sigma_0^2 = 100\), confirmando la no significación del contraste bilateral.

¿Qué calcula esta herramienta?

Esta calculadora de contraste de hipótesis para una varianza aplica el test chi-cuadrado de una muestra sobre la varianza poblacional. Proporciona el estadístico \(\chi^2 = (n-1)s^2/\sigma_0^2\), los grados de libertad, el p-valor adaptado al tipo de contraste (bilateral, cola derecha o cola izquierda) y el intervalo de confianza para la varianza \(\sigma^2\) y la desviación típica \(\sigma\) al nivel elegido. Las entradas son la desviación típica muestral \(s\) y el valor de referencia \(\sigma_0\) (no las varianzas directamente).

Preguntas frecuentes

  • ¿Por qué el test chi-cuadrado de varianza es sensible a la no normalidad? Porque la distribución \(\chi^2(n-1)\) de \((n-1)s^2/\sigma^2\) depende de la normalidad de la población; datos no normales pueden producir distribuciones muy diferentes de \(s^2\), invalidando el p-valor.
  • ¿Cuándo usar el contraste unilateral derecho? Cuando se desea detectar si la variabilidad ha aumentado por encima de un límite de especificación, habitual en control de calidad o evaluación de precisión de instrumentos.
  • ¿Cómo se interpreta el IC para σ? Proporciona el rango de valores de \(\sigma\) compatibles con los datos al nivel de confianza elegido; si \(\sigma_0\) queda fuera del IC bilateral, coincide con el rechazo de \(H_0\) en el contraste bilateral.
  • ¿Qué diferencia hay entre este test y el test F de dos varianzas? Este test compara la varianza de una muestra frente a un valor fijo \(\sigma_0^2\); el test F compara las varianzas de dos muestras independientes entre sí.
