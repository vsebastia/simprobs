¿Qué es un contraste de hipótesis?

Es un procedimiento de inferencia para evaluar la compatibilidad de los datos con una hipótesis nula frente a una alternativa.

contraste de hipótesis para una media

Guía rápida

  • Si contrastas una media, usa las herramientas de una media o dos medias.
  • Si comparas proporciones, usa los contrastes de una o dos proporciones.
  • Si trabajas con variables categóricas, usa chi-cuadrado dentro de distribuciones y contrastes apropiados.
