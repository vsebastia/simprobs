¿Qué es un contraste de hipótesis?

Un contraste de hipótesis es un procedimiento estadístico para decidir, a partir de datos muestrales, si la evidencia es suficiente para rechazar una afirmación inicial sobre una población.

Hipótesis nula y alternativa

  • Hipótesis nula (H₀): representa el escenario de referencia (por ejemplo, “no hay diferencia” o “el parámetro vale un valor concreto”).
  • Hipótesis alternativa (H₁): plantea el efecto o diferencia que se quiere detectar.

Errores tipo I y tipo II

  • Error tipo I (α): rechazar H₀ siendo verdadera (falso positivo).
  • Error tipo II (β): no rechazar H₀ siendo falsa (falso negativo).
  • Potencia (1-β): probabilidad de detectar un efecto real.

Estadístico de contraste y regiones de decisión

El estadístico de contraste resume la evidencia de la muestra (por ejemplo, un estadístico z o t). A partir de su distribución bajo H₀ se define una región de rechazo y una región de no rechazo (a veces llamada de aceptación en un sentido operativo). Si el estadístico cae en la región de rechazo, se rechaza H₀; si no, no se rechaza.

El p-valor cuantifica cuán extremos son los datos observados bajo H₀: cuanto menor es, mayor evidencia contra H₀.
