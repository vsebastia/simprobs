Explicación breve

El test exacto de Fisher evalúa si existe asociación entre dos variables binarias en una tabla 2×2. Es especialmente apropiado cuando las frecuencias esperadas en alguna celda son inferiores a 5, situación en la que el test chi-cuadrado de independencia no es fiable.

La potencia exacta se calcula como la suma de probabilidades de todos los pares de recuentos \((a, c)\) que producirían un p-valor de Fisher ≤ α, ponderados por su probabilidad bajo \(H_1\):

\( \text{Potencia} = \sum_{a=0}^{n}\sum_{c=0}^{n} P(a \mid p_1) \cdot P(c \mid p_2) \cdot \mathbf{1}[\text{p-valor Fisher}(a,n-a,c,n-c) \leq \alpha] \)

donde \(P(a \mid p_1) = \binom{n}{a} p_1^a (1-p_1)^{n-a}\) sigue una distribución binomial. Para \(n > 150\) se aplica la aproximación chi-cuadrado de dos proporciones.

Fórmula del p-valor exacto de Fisher

Para una tabla 2×2 con recuentos \((a, b, c, d)\) y marginales fijos \((n_1 = a+b,\ n_2 = c+d,\ m = a+c,\ N = n_1+n_2)\), el p-valor exacto bilateral suma las probabilidades hipergeométricas menores o iguales a la observada:

\( p = \sum_{x:\, P_{\text{hiper}}(x) \leq P_{\text{hiper}}(a)} P_{\text{hiper}}(x) \quad \text{donde} \quad P_{\text{hiper}}(x) = \frac{\binom{m}{x}\binom{N-m}{n_1-x}}{\binom{N}{n_1}} \)

¿Cuándo usar el test exacto de Fisher?

  • Muestras pequeñas con frecuencias esperadas < 5 en alguna celda.
  • Tablas dispersas (sparse): datos con muchos ceros.
  • Estudios caso-control con resultados raros (baja prevalencia).
  • Siempre que se quiera garantizar el control exacto del error tipo I.

Ejemplo resuelto

Un ensayo clínico compara la tasa de respuesta de dos tratamientos: se espera \(p_1 = 0{,}30\) en el grupo control y \(p_2 = 0{,}50\) en el grupo intervención. Con \(\alpha = 0{,}05\) y potencia = 80 %, la calculadora devuelve aproximadamente n ≈ 83 por grupo (total 166 observaciones). Para ese tamaño de muestra, si la diferencia real es de 20 puntos porcentuales, el test detectará la diferencia el 80 % de las veces.

Nota: si la diferencia de proporciones es inferior a 0,05 (muy pequeña), se necesitarán muestras muy grandes y la convergencia puede ser lenta.

¿Qué calcula esta herramienta?

Encuentra por búsqueda iterativa el menor \(n\) (por grupo) tal que la potencia incondicional del test exacto de Fisher bilateral alcanza el valor objetivo. Para \(n \leq 150\) se usa la potencia exacta basada en la distribución hipergeométrica; para \(n > 150\) se aplica la aproximación chi-cuadrado (test de dos proporciones con varianza agrupada).

Cómo interpretar el resultado

El valor obtenido es el mínimo teórico por grupo asumiendo diseño balanceado. Si prevés pérdidas, divide \(n\) entre \((1 - \text{tasa de pérdida})\) para obtener el tamaño de reclutamiento.
