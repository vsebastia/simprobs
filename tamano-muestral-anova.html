Explicación breve

El ANOVA de un factor compara las medias de k ≥ 2 grupos independientes. Para planificar la muestra necesitas el número de grupos, el tamaño del efecto f de Cohen (cociente entre la variabilidad entre medias y la variabilidad residual), el nivel de significación \(\alpha\) y la potencia objetivo \(1-\beta\).

Fórmula y método de cálculo

Con n observaciones por grupo:

\( \lambda = n \cdot k \cdot f^2 \)

El estadístico F bajo \(H_1\) sigue una distribución F no central con parámetros \(\text{gl}_1 = k-1\), \(\text{gl}_2 = k(n-1)\) y parámetro de no centralidad \(\lambda\). La potencia es:

\( 1-\beta = P\!\left(F'(k-1,\,k(n-1),\,\lambda) > F_{\alpha,\,k-1,\,k(n-1)}\right) \)

Se busca el mínimo n entero tal que la potencia supere el objetivo.

  • f: efecto de Cohen = \(\sigma_{\text{medias}} / \sigma_{\text{residual}}\).
  • λ: parámetro de no centralidad; crece con n, k y f.
  • Referencia: f = 0,10 (pequeño), 0,25 (mediano), 0,40 (grande).

Configuración rápida

  • f desconocido: usa f = 0,25 (mediano) si no tienes datos previos.
  • f desde datos: calcula \(\sigma_m\) (std de las medias esperadas) y \(\sigma\) (std residual); f = σ_m/σ.
  • Alfa: 0,05 es el valor habitual; bájalo a 0,01 en estudios confirmatorios.
  • Potencia: 0,80 como mínimo; 0,90 si el estudio es crítico.

Ejemplo sencillo

Quieres comparar 3 dietas con un efecto mediano (f = 0,25), α = 0,05 y potencia 0,80. El resultado es aproximadamente 52 sujetos por grupo (total ≈ 156).

Usos frecuentes

  • Comparación de 3 o más tratamientos en ensayos experimentales.
  • Diseños con varios niveles de un factor (dosis, condición, centro).
  • Planificación de estudios donde se prevé un análisis post-hoc de medias.

¿Qué calcula esta herramienta?

Esta calculadora estima el tamaño de muestra mínimo por grupo para un ANOVA de un factor balanceado con k grupos, dado el tamaño del efecto de Cohen (f), el nivel de significación y la potencia objetivo.

Fórmula utilizada

Se busca el mínimo n tal que la potencia de la F no central supera el objetivo. La potencia se calcula mediante la distribución F no central con parámetro de no centralidad λ = n·k·f².

Ejemplo de uso

Introduce k = 3 grupos, f = 0,25 (efecto mediano), α = 0,05 y potencia = 0,80. El resultado indica cuántos sujetos necesitas por grupo para detectar ese efecto con esa fiabilidad.

Cómo interpretar el resultado

El n calculado es por grupo (diseño balanceado). Multiplícalo por k para obtener el total. Si esperas pérdidas, divide por (1 − tasa de abandono esperada).

Preguntas frecuentes

  • ¿Qué es f de Cohen? El cociente entre la std de las medias grupales y la std residual. Valores: 0,10 pequeño, 0,25 mediano, 0,40 grande.
  • ¿Funciona para ANOVA con grupos desiguales? Esta calculadora asume diseño balanceado (igual n por grupo). Para grupos desiguales ajusta con el n armónico.
  • ¿Qué es la potencia estadística? La probabilidad de detectar un efecto real cuando existe.
  • ¿El tamaño muestral es exacto? No; depende de los supuestos y se redondea al entero superior.
