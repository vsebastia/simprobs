Explicación breve

El test de Kolmogorov–Smirnov de una muestra (bondad de ajuste) compara la función de distribución empírica \(F_n(x)\) con una función de distribución teórica completamente especificada \(F_0(x)\). El estadístico es la distancia máxima entre ambas:

\( D_n = \sup_{x} \left| F_n(x) - F_0(x) \right| \)

La aproximación asintótica de la potencia se basa en la distribución de Kolmogorov. Para muestras pequeñas (n < 30) esta aproximación puede sobreestimar ligeramente la potencia real.

Fórmula de potencia (aproximación asintótica)

El valor crítico se obtiene invirtiendo la distribución de Kolmogorov: \(c_\alpha = K^{-1}(1-\alpha)\), donde \(K(t) = 1 - 2\sum_{j=1}^{\infty}(-1)^{j-1}e^{-2j^2t^2}\). La potencia para un efecto δ es:

\( \text{Potencia} \approx 1 - K\!\left(c_\alpha - \sqrt{n}\,\delta\right) + K\!\left(-c_\alpha - \sqrt{n}\,\delta\right) \)

La calculadora busca el menor \(n\) tal que la potencia alcanza el objetivo, con \(n\) entre 2 y 50 000.

Tamaños de efecto de referencia

  • δ = 0,10: efecto pequeño (la CDF empírica se desvía como máximo 10 pp de la teórica).
  • δ = 0,20: efecto moderado (referencia habitual en estudios de bondad de ajuste).
  • δ = 0,30: efecto grande (desviación muy apreciable).

Extensión al test KS de dos muestras

Para el test KS de dos muestras con grupos de igual tamaño, el tamaño efectivo de muestra es \(n_{\text{ef}} = n_1 \cdot n_2 / (n_1 + n_2)\). Con grupos balanceados, \(n_{\text{ef}} = n/2\), por lo que cada grupo necesita el doble del \(n\) calculado aquí.

Ejemplo resuelto

Un investigador quiere detectar una desviación máxima \(\delta = 0{,}20\) entre la distribución empírica y una distribución exponencial teórica. Con \(\alpha = 0{,}05\) y potencia = 80 %, la calculadora devuelve aproximadamente n ≈ 52 observaciones.

Advertencia: si δ es muy pequeño (< 0,05) se pueden necesitar miles de observaciones. Para muestras de n < 8 la aproximación asintótica puede sobreestimar la potencia real.

¿Qué calcula esta herramienta?

Encuentra por búsqueda iterativa el menor \(n\) tal que la potencia asintótica del test de Kolmogorov–Smirnov de una muestra alcanza el valor objetivo para un efecto δ dado. La aproximación es exacta para n grande; para n < 8 se muestra una advertencia.

Cómo interpretar el resultado

El valor devuelto es el número mínimo de observaciones necesarias. El valor crítico aproximado \(D_{n,\alpha} \approx c_\alpha / \sqrt{n}\) permite saber qué magnitud de desviación resulta estadísticamente significativa con ese tamaño de muestra.
