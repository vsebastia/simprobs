Descripción

Este contraste se utiliza para evaluar si dos medias poblacionales son iguales cuando se comparan dos grupos independientes y no se asume varianza común.

\(H_0: \mu_1 - \mu_2 = 0\)

\( t = \frac{\bar{x}_1 - \bar{x}_2}{\sqrt{s_1^2/n_1 + s_2^2/n_2}} \)

Conceptos clave del contraste

Un contraste de hipótesis ayuda a decidir si la diferencia observada entre grupos puede explicarse por azar muestral o si apunta a una diferencia real en la población.

  • Error tipo I (\(\alpha\)): probabilidad de concluir que las medias difieren cuando en realidad no difieren.
  • Error tipo II (\(\beta\)): probabilidad de no detectar una diferencia real entre medias.
  • Potencia (\(1-\beta\)): probabilidad de detectar una diferencia real de interés. Aumenta con más tamaño muestral, menor variabilidad o mayor efecto esperado.

¿Por qué aquí se usa Welch?

Se utiliza la fórmula de Welch porque no obliga a suponer varianzas iguales entre grupos. En práctica aplicada, es una opción robusta cuando \(s_1\) y \(s_2\), o \(n_1\) y \(n_2\), son distintos.

Por eso también se usan grados de libertad aproximados (Welch-Satterthwaite), que pueden no ser enteros.

¿Qué calcula esta herramienta?

Esta calculadora devuelve el estadístico de contraste, p-valor y una conclusión básica para decidir si los datos son compatibles con la hipótesis nula.

Fórmula utilizada

Se aplica el estadístico de prueba específico del contraste y se transforma en un p-valor según cola bilateral o unilateral y nivel de significación.

Ejemplo de uso

Define hipótesis nula y alternativa, introduce los datos muestrales y selecciona α. Al calcular, obtendrás p-valor y regla de decisión para el contraste.

Cómo interpretar el resultado

Si p-valor < α, se rechaza H0. Si p-valor ≥ α, no hay evidencia suficiente para rechazar H0. La significación estadística no implica relevancia práctica.

Preguntas frecuentes

  • ¿Qué es la hipótesis nula? Es la afirmación de referencia que se intenta poner a prueba.
  • ¿Qué representa el p-valor? La compatibilidad de los datos observados con H0 bajo el modelo del contraste.
  • ¿Qué diferencia hay entre bilateral y unilateral? El bilateral evalúa diferencias en ambos sentidos y el unilateral en una sola dirección.
  • ¿Rechazar H0 implica efecto importante? No necesariamente; también debes valorar tamaño de efecto y contexto aplicado.
