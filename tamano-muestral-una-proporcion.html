Tamaño muestral

Una proporción

Calcula el tamaño mínimo de muestra para estimar una proporción con un margen de error deseado.

Volver al índice de tamaños muestrales

Descripción

Este cálculo se usa cuando quieres estimar una única proporción poblacional (por ejemplo, porcentaje de clientes satisfechos) con una precisión concreta. Necesitas una proporción esperada p, un margen de error absoluto E y un nivel de confianza para obtener el valor crítico Z.

Fórmula de tamaño muestral

\( n = \frac{Z^2 \cdot p \cdot (1-p)}{E^2} \)

  • n: tamaño muestral mínimo.
  • Z: cuantil normal para el nivel de confianza (ej. 1.96 para 95%).
  • p: proporción esperada.
  • E: margen de error máximo permitido.
