Sobre ProbLab

ProbLab reúne calculadoras estadísticas gratuitas, explicaciones y ejemplos para aprender y aplicar inferencia estadística desde el navegador.

Volver al inicio

Qué es ProbLab

ProbLab es una herramienta web educativa creada para estudiantes, docentes, analistas y cualquier persona que necesite calcular probabilidades, intervalos de confianza, tamaños muestrales, contrastes de hipótesis o resultados de test A/B sin instalar software adicional.

El objetivo es combinar cálculo interactivo con explicación clara: cada herramienta prioriza fórmulas visibles, supuestos, interpretación de resultados y ejemplos aplicados.

Quién está detrás

El proyecto está mantenido por Vicente Castellar. Puedes contactar para correcciones, sugerencias o propuestas de mejora en vic.sebastia@gmail.com.

ProbLab no sustituye el criterio profesional ni la revisión metodológica de un estudio, pero ofrece una base práctica para verificar cálculos y comprender los pasos estadísticos habituales.

Qué encontrarás en la web

  • Calculadoras de distribuciones para PDF, CDF, colas y cuantiles.
  • Tablas estadísticas con valores críticos y probabilidades acumuladas.
  • Tamaños muestrales para medias, proporciones y comparaciones.
  • Intervalos de confianza para estimaciones e incertidumbre.
  • Contrastes de hipótesis para medias, proporciones, varianzas y bondad de ajuste.
  • Herramientas de test A/B para conversión, potencia, enfoque bayesiano y simulación.

Transparencia y mejoras

Las calculadoras se revisan para que las fórmulas, los valores críticos y las interpretaciones sean coherentes. Si detectas una discrepancia, indícala junto con el caso de entrada y el resultado esperado para poder reproducirla.

Consulta también la página de metodología para conocer los criterios de cálculo, redondeo y limitaciones.
