Política de privacidad

Esta política explica qué datos pueden tratarse al navegar por esta web y con qué finalidad.

Datos solicitados directamente

Esta web no dispone de formularios, no permite crear cuentas de usuario y no solicita datos personales directamente al visitante.

Datos técnicos de navegación

Durante la navegación pueden generarse datos técnicos necesarios para el funcionamiento, seguridad y mantenimiento del sitio web. Estos datos pueden incluir la dirección IP, fecha y hora de acceso, URL solicitada, país aproximado de origen, navegador, dispositivo, sistema utilizado, tipo de petición, códigos de respuesta del servidor e información técnica asociada a errores o seguridad.

Estos datos pueden ser tratados por el proveedor de alojamiento web con la finalidad de prestar el servicio de hosting, garantizar la seguridad, prevenir abusos y mantener el correcto funcionamiento técnico del sitio.

Proveedor de alojamiento

Esta web está alojada en servidores de Hostinger. El proveedor de alojamiento puede tratar registros técnicos del servidor en el marco de la prestación del servicio.

Google Search Console y Bing Webmaster Tools

Esta web puede estar verificada en Google Search Console y Bing Webmaster Tools. Estas herramientas permiten consultar información agregada sobre la presencia del sitio en buscadores, como consultas, impresiones, clics, páginas indexadas, errores de rastreo y rendimiento SEO.

El responsable del sitio no utiliza estas herramientas para identificar personalmente a los visitantes.

Finalidad del tratamiento

Los datos técnicos pueden tratarse para permitir el funcionamiento de la web, garantizar la seguridad, prevenir abusos, analizar errores técnicos y mejorar la estructura, el contenido y la indexación del sitio.

Base jurídica

La base jurídica para el tratamiento de datos técnicos necesarios para el funcionamiento, seguridad y mantenimiento del sitio web es el interés legítimo del responsable en mantener la web disponible, segura y correctamente indexada.

Conservación

Los registros técnicos se conservarán durante el tiempo necesario para cumplir las finalidades indicadas. Los periodos concretos de conservación pueden depender de la configuración y políticas del proveedor de alojamiento.

Derechos de los usuarios

Esta web no identifica directamente a los visitantes ni dispone de cuentas de usuario. Por ello, normalmente no es posible asociar los datos técnicos de navegación a una persona concreta.

Cuando resulte aplicable, los usuarios pueden ejercer los derechos previstos en la normativa de protección de datos escribiendo a vic.sebastia@gmail.com. Para poder atender la solicitud, puede ser necesario facilitar información que permita localizar los datos técnicos correspondientes, como la dirección IP, fecha aproximada de acceso o URL visitada.

Los usuarios también pueden presentar una reclamación ante la Agencia Española de Protección de Datos si consideran que el tratamiento de sus datos no se ajusta a la normativa aplicable.
