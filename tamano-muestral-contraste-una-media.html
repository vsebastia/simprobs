Explicación breve

Esta calculadora estima el tamaño muestral para un contraste de una media: \(H_0: \mu=\mu_0\) frente a alternativas bilateral o unilateral.

Fórmula utilizada

\( n = ((Z_{lpha^*}+Z_{eta})\sigma/\Delta)^2 \)

Con \(lpha^*=lpha/2\) para Ha: \(\mu eq\mu_0\), y \(lpha^*=lpha\) para Ha: \(\mu\geq\mu_0\) o Ha: \(\mu\leq\mu_0\).

  • \(\sigma\): desviación estándar esperada.
  • \(\Delta\): diferencia mínima detectable \(|\mu-\mu_0|\).
  • \(1-eta\): potencia objetivo.

Ejemplo

Si \(\sigma=10\), \(\Delta=2\), \(lpha=0{,}05\), potencia \(0{,}80\) y contraste bilateral, el tamaño muestral es cercano a 196 observaciones.
